# Generative AI & LLM Engineering Projects

A collection of projects demonstrating the design, development, and deployment of modern Generative AI applications using both frontier and open-source Large Language Models (LLMs).

The repository showcases practical implementations of contemporary LLM engineering techniques, including Retrieval-Augmented Generation (RAG), fine-tuning with QLoRA, AI agents, function calling, multimodal applications, vector search, and production-ready AI systems.

## Featured Projects

* 🌐 AI-powered website brochure generator with intelligent web scraping
* ✈️ Multimodal customer support assistant with tool calling and conversational UI
* 🎙️ Speech-to-text meeting assistant with automated summaries and action items
* 💻 AI-assisted Python-to-C++ code optimization for high-performance computing
* 📚 Enterprise knowledge assistant powered by RAG and vector databases
* 💰 Product price prediction using frontier and fine-tuned open-source models
* 🤖 Autonomous AI agent for deal discovery and automated notifications

This repository reflects practical implementations of modern LLM engineering patterns and AI application development. Each project focuses on solving real-world problems using state-of-the-art Generative AI techniques, ranging from intelligent assistants and knowledge retrieval systems to code generation, multimodal AI, and autonomous agent workflows.
